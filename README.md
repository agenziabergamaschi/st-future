# Il futuro di Spazio Tricolore

Questa repository si propone di raccogliere e tracciare le novità relative allo sviluppo dei servizi di Spazio Tricolore:
- [segnalazioni di problemi](https://github.com/agenziabergamaschi/st-future/issues?q=sort%3Aupdated-desc%20is%3Aissue%20is%3Aopen%20type%3ABug);
- [richieste di nuove funzionalità o servizi](https://github.com/agenziabergamaschi/st-future/issues?q=sort%3Aupdated-desc%20is%3Aissue%20is%3Aopen%20type%3AFeature) da parte degli utenti;
- [rilasci di aggiornamenti o nuovi servizi](https://github.com/agenziabergamaschi/st-future/releases).

Se vuoi una vista d'insieme sui nuovi servizi o funzionalità in arrivo, consulta [il progetto](https://github.com/orgs/agenziabergamaschi/projects/4).

Se hai riscontrato un problema o vorresti richiedere nuove funzionalità, per favore [controlla se vi sono già segnalazioni al riguardo](https://github.com/agenziabergamaschi/st-future/issues?q=sort%3Aupdated-desc%20is%3Aissue%20is%3Aopen) e, in caso negativo, [apri un ticket (“_issue_”)](https://github.com/agenziabergamaschi/st-future/issues/new/choose).

In alternativa, [scrivici](mailto:info@spaziotricolore.it)!
